# E-Commerce Customer Behaviour Analysis Dashboard

An interactive Tableau dashboard that analyzes customer behavior, demographics, spending patterns, membership performance, and customer ratings.

##  Tools Used
- Tableau
- Excel

##  Dashboard Highlights
- Customer Distribution by City
- Age-wise Spending Analysis
- Gender Distribution
- Membership Analysis
- Customer Ratings
- Key Business KPIs

##  Files
- `Dashboard.twbx` – Tableau dashboard
- `Dataset.xlsx` – Source dataset
- `Dashboard.png` – Dashboard preview

##  Skills Demonstrated
- Data Visualization
- Business Intelligence
- Dashboard Design
- KPI Analysis
- Exploratory Data Analysis (EDA)
